# Alphabet River Runner 🎮

Alphabet River Runner is a 3-lane educational runner game built in Unity (3D Core).  

The objective is to jump across floating lotus platforms and collect letters in alphabetical order from \*\*A to Z\*\*.

## 🚀 Game Concept

The player moves across a river by jumping between lotus platforms arranged in three lanes.  

Each lotus contains a letter. The player must land on the correct next alphabet letter.

- Correct jump → Letter disappears and game continues

- Wrong jump → Player falls into the river (Game Over)

- Reach Z → You Win

---

## 🎮 Controls


| Key | Action |

|------|--------|

| ← | Move Left Lane |

| → | Move Right Lane |

| ↑ | Jump Forward |
